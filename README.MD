# Gestión de Repositorios con Git y GitHub

## Descripción
[cite_start]Este proyecto simula un flujo de trabajo colaborativo para el aprender a utilizar GIT & GITHUB. 
Se implementaron funcionalidades básicas de HTML gestionadas mediante el sistema de control de versiones Git.

## Funcionalidades Implementadas
* [cite_start]**Estructura Base:** Creación de archivos `index.html` y `README.md`
* [cite_start]**Barra de Navegación:** Implementada en la rama `feature/navbar`
* [cite_start]**Pie de Página:** Implementado mediante Pull Request en la rama `feature/footer`

## Ramas Utilizadas
* [cite_start]`main`: Rama principal de producción.
* [cite_start]`feature/navbar`: Rama para el desarrollo del encabezado.
* [cite_start]`feature/footer`: Rama para el desarrollo del pie de página.
